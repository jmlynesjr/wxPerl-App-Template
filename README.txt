Simple shell script to create basic wxPerl applications
    Modify the template files as needed and run sh wx-newapp.sh

James M. Lynes, Jr May 4, 2016

------------------------------------------------------------------------

wx-app-top-template           - Source file header template and app setup
wx-titletext-template         - Define title text for the frame
wx-button-template            - Define some buttons
wx-dataentrybox-template      - Define a data entry box
wx-messagebox-template        - Define a message box
wx-radiobox-template          - Define a radio box
wx-sizer-template             - Define a "Standard" layout
wx-event-template             - Define some events
wx-timer-template             - Define a timer
wx-app-bottom-template        - Complete the source file
wx-newapp.sh                  - Merges all of the templates
wx-newapp.pl                  - Merged templates
